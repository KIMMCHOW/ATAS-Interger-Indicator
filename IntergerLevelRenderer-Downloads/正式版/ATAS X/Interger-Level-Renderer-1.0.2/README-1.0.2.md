# Interger Level Renderer 1.0.2

English: Interger Level Renderer draws 100-unit and 1000-unit integer price levels on the ATAS main chart. It does not place orders, change trading state, or write to ATAS installation files.

中文：Interger Level Renderer 会在 ATAS 主图绘制 100 单位和 1000 单位整数价格关口。它不下单、不改变交易状态，也不写入 ATAS 安装目录。

## Install / 安装

English: Copy `Interger-Level-Renderer-1.0.2-ATAS-X.dll` to `%APPDATA%\ATAS\Indicators`, restart ATAS, and search for `Interger Level Renderer`.

中文：将 `Interger-Level-Renderer-1.0.2-ATAS-X.dll` 复制到 `%APPDATA%\ATAS\Indicators`，重启 ATAS，然后搜索 `Interger Level Renderer`。
