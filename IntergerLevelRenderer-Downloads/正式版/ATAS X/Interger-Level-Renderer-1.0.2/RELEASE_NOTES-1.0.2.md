# Release Notes 1.0.2 / 版本说明 1.0.2

English:

- Current stable TradingHub release for ATAS X.
- Aligns assembly and file-version metadata with the published product version.
- Renders 100-unit and 1000-unit levels on the main chart.
- Includes a quiet TradingHub latest-version check that only shows one small update line when a newer version exists.

中文：

- 当前面向 ATAS X 的 TradingHub 稳定版本。
- 将 assembly 和文件版本元数据与已发布产品版本对齐。
- 在主图绘制 100 单位和 1000 单位整数关口。
- 包含静默 TradingHub 最新版本检查；只有发现新版本时才显示一行小号提示。
